# Full-Stack Web Development & AI Engineering Bootcamp

Welcome to my comprehensive 6-week Full-Stack MERN Stack and AI Integration blueprint. This repository acts as a live ledger of my day-by-day progression from basic engineering architectures to enterprise-ready system workflows, containerization, and advanced generative AI implementation.

---

## 🛠️ Tech Stack & Tooling Core
* **Frontend:** React.js, Vite, Tailwind CSS, React Router, Context API
* **Backend:** Node.js, Express.js, Model-View-Controller (MVC) Architecture
* **Databases & BaaS:** MongoDB (Mongoose ODM), Supabase (PostgreSQL), Redis
* **AI Ecosystem:** OpenAI API, Gemini API, Prompt Engineering, Vector Embeddings
* **DevOps & Architecture:** Docker, GitHub Actions (CI/CD), Apache Kafka, System Design
